# Plan Interactivo — Data Analyst

## Qué es esto

Un plan de estudio autoguiado y navegable en el browser: fundamentos, herramientas
y skills que se piden hoy para un perfil de Data Analyst, organizado en etapas para
que puedas ir marcando tu progreso.

No hace falta instalar nada — se abre directo desde el link de arriba.

## Cómo usarlo

Entrá al link, es una sola página interactiva. Funciona en desktop y mobile.

## Sobre Mermelada Tech

Comunidad de mujeres en tecnología: mentorías, contenido educativo y acompañamiento
para quienes están empezando o creciendo en el rubro.
